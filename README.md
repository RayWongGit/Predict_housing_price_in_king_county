## About this project
This project is the assignment of the [Data Analysis with Python](https://www.coursera.org/learn/data-analysis-with-python) from the [IBM Data Science Professional Certificate](https://www.coursera.org/professional-certificates/ibm-data-science).

## What do we do in this project

In this project:

- we wrangle the data
- do exploratory data analysis on the cleaned data 
- build linear regerssion and Ridge regression  models to predict housing prices, using features such as square footage, number of bedrooms, number of floors and so on. 
- then evualate how good our models can predict
